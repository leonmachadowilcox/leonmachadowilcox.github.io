Animal Shelter Dashboard

Refactor to Flask blueprints + service/model layers, improved UX, and secure forms.

← Back to Artifacts View Source

Overview

This enhancement separates routing, business logic, and data access using Flask Blueprints and a simple service/model layer. It also standardizes forms, adds CSRF protection, and cleans up the UI to align with accessibility and UX best practices.

Security & Configuration

  • CSRF: enabled via Flask-WTF for all POST forms.
  • Validation: server-side schema & field checks for user input.
  • Sessions: secure cookies; login-required routes for admin actions.
  • Config: separate config.Dev and config.Prod with environment variables for secrets.
